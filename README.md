# prog_to_latex
Pseudo-code to LaTeX code

## Operators
- `frac` : $/$
- `int` : $\int$
- `sum` : $\sum$
- `prod` : $\prod$
- `lim` : $\lim$
- `<=` : $\leqslant$ 
- `>=` : $\geqslant$
- `==>` : $\implies$
- `<==>` : $\iff$
- `in` : $\in$
- `inc` : $\subset$
- `=` : $=$
- `+` : $+$
- `-` : $-$
- `*` : $\cdot$
- `/` : $/$
- `nN` : $\mathbb{N}$
- `zZ` : $\mathbb{Z}$
- `qQ` : $\mathbb{Q}$
- `rR` : $\mathbb{R}$
- `cC` : $\mathbb{C}$
- `forall` : $\forall$
- `exists` : $\exists$
- `exists!` : $\exists !$

## Syntax 
To quit the program, type `quit` and press `Enter`.
