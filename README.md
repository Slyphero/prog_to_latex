# prog_to_latex
Pseudo-code to LaTeX code

## Operators
- `frac` : $\frac{...}{...}$
- `int` : $\int$
- `sum` : $\sum$
- `prod` : $\prod$
- `lim` : $\lim$
- `<=` : $\leq$ 
- `>=` : $\geq$
- `==>` : $\implies$
- `<==>` : $\iff$
- `in` : $\in$
- `subset` : $\subset$
- `=` : $=$
- `rR` : $\mathbb{R}$
- `cC` : $\mathbb{C}$
- `zZ` : $\mathbb{Z}$
- `nN` : $\mathbb{N}$
- `qQ` : $\mathbb{Q}$

## Syntax 
To quit the program, type `quit` and press `Enter`.
