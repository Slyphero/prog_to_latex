# prog_to_latex
Pseudo-code to LaTeX code

## Syntax 
To quit the program, type `quit` and press `Enter`.  

- `int a b f(x) x` : $\int_a^b f(x) \ \mathrm{d}x$
- `sum k=0 n a_k` : $\sum_{k=0}^n a_k$
- `prod` : $\prod$
- `lim` : $\lim$
- `a <= b` : $a \leqslant b$ 
- `a >= b` : $a \geqslant b$
- `P ==> Q` : $P \implies Q$
- `<==>` : $P \iff Q$
- `x in E` : $x \in E$
- `A inc B` : $A \subset B$
- `=` : $=$
- `+` : $+$
- `-` : $-$
- `*` : $\cdot$
- `a/b` : $\frac{a}{b}$
- `a^b` : $a^b$
- `a_b` : $a_b$
- `nN` : $\mathbb{N}$
- `zZ` : $\mathbb{Z}$
- `qQ` : $\mathbb{Q}$
- `rR` : $\mathbb{R}$
- `cC` : $\mathbb{C}$
- `forall` : $\forall$
- `exists` : $\exists$
- `exists!` : $\exists !$
- `sup` : $\sup$
- `inf` : $\inf$
