# prog_to_latex
Pseudo-code to LaTeX code

## Operators
- `frac` : $\frac{...}{...}$
- `int` : $\int$
- `sum` : $\sum$
- `prod` : $\prod$
- `lim` : $\lim$
- `<=` : $\leq$ 
- `>=` : $\geq$
- `==>` : $\implies$
- `<==>` : $\iff$
- `in` : $\in$
- `subset` : $\subset$
- `=` : $=$

## Syntax 
To quit the program, type `quit` and press `Enter`.
